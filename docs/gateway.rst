搭建网关：透明代理、自动分流
==================================

适用 CN 墙内，使以下形式共存：

- ``socks5 代理``
- ``socks5 代理`` 自动分流
- ``HTTP 代理``
- ``HTTP 代理`` 自动分流
- ``网关`` 自动分流


局域网内所有设备，自动分流

- 代理：其它 DNS 解析和访问流量
- 直连：局域网流量
- 直连：墙内 DNS 解析和访问流量
- 直连：BT 流量
- 拦截：广告流量

- B
